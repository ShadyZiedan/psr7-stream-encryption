# Скрипты

Полезные скрипты для работы с проектом.

## setup.sh

Автоматическая настройка проекта:

1. Проверка Docker и Docker Compose
2. Сборка образа
3. Установка зависимостей
4. Проверка требований PHP
5. Запуск тестов
6. Запуск примера

### Использование

```bash
chmod +x scripts/setup.sh
./scripts/setup.sh
```

### Требования

- Docker
- Docker Compose
- Make (опционально)
